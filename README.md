# Dr. Ben Romdhane Brahim - Ophthalmology Landing Page

This is a landing page project built with Next.js and Tailwind CSS for Dr. Ben Romdhane Brahim, an ophthalmologist based in Mahdia, Tunisia.

## Features

- Modern, responsive design using Tailwind CSS
- Multi-section presentation of medical services, eye exams, and treatments
- French language content tailored for local patients
- SEO friendly and fast server-side rendering with Next.js

## Technologies Used

- [Next.js](https://nextjs.org/) - React framework with server-side rendering
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework for styling
